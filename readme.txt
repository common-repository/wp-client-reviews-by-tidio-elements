Reviews from your clients displayed as a widget in the lower right corner of your site.

== Description ==

Plugin allowing the users of your site to get acquainted with your client reviews in a simple and transparent way. 
The reviews will be displayed in the form of a widget placed in the lower right corner of the site. After clicking on the widget a list of client reviews will show up.
In addition, users will be able to add reviews right from your website, or you'll be able to add reviews right from the WordPress administration panel.

[youtube https://www.youtube.com/watch?v=SRf7W5bRbLI]

= Features: =
* Management panel of client reviews 
* Possibility of adding client reviews straight from the website 
* Possibility to rate reviews from 1 to 5 
* Plugin immediately available after installation on your site 
* Support for multiple languages

= Notice: =
Plugin "Customer Reviews" is no longer supported, if you're having any issues with the basic functionality of plugin, please contact us on contact@tidio.net

== Installation ==
1. Go to WordPress Control Panel
1. Click "Plugins", then "Add New"
1. Type in "tidio clients reviews" and click "Search Plugins"
1. Download and install the Plugin
1. Click the "Activate Plugin" link

== Frequently Asked Questions ==

= Is the use of the plugin free?  =
Yes, the use of the plugin is completely free! 

= Where are stored the client reviews I have added?  =
All data that you have added are stored on the servers of the plugin provider - tidioelements.com 

= How can I manage the list of added reviews?  =
To manage the list of questions , after installing the plugin go to the "Client Reviews" tab in your admin panel
